# Responsive Tech Webapp

A separate responsive-first portfolio style webapp with project repository cards, preview images, and tech compatibility sections.

## Features

- Mobile, tablet, laptop, and desktop responsive behavior
- Tech stack filters
- Multiple project cards with source links and in-app live preview routes
- About section for project positioning and stack coverage
- Contact section with direct details and a ready-to-use message form
- Clean modern UI with fast Vite setup

## Run

```powershell
npm install
npm run dev
```

Open http://localhost:5180

## Build

```powershell
npm run build
npm run preview
```
