# Responsive Tech Webapp / เว็บแอปสายเทคแบบ Responsive

English:

A separate responsive-first portfolio style webapp with project repository cards, live demo links, in-app quick previews, and tech compatibility sections.

ไทย:

โปรเจกต์เว็บแอปแยกที่ออกแบบแบบ responsive-first มีการ์ดโปรเจกต์พร้อมลิงก์ repo, ลิงก์ live demo ภายนอก, quick preview ภายในหน้า และส่วนแสดงความครอบคลุมด้านเทคโนโลยี

## Features / ความสามารถหลัก

- Mobile, tablet, laptop, and desktop responsive behavior / รองรับการใช้งานบนมือถือ แท็บเล็ต แล็ปท็อป และเดสก์ท็อป
- EN/TH language switch on the UI / มีสวิตช์ภาษาอังกฤษและไทยในหน้าเว็บ
- Tech stack filters / มีระบบกรองโปรเจกต์ตาม tech stack
- Live Demo external links (StackBlitz) / มีลิงก์ Live Demo ภายนอก (StackBlitz)
- In-app Quick Preview routes (`#preview/...`) / มี Quick Preview ภายในหน้า (`#preview/...`)
- About and Contact sections / มีส่วน About และ Contact พร้อมใช้งาน

## Run / วิธีรัน

```powershell
npm install
npm run dev
```

Open: http://localhost:5180

## Build / วิธี build

```powershell
npm run build
npm run preview
```
